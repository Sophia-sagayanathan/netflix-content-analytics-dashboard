# Netflix Content Analytics Dashboard

An interactive Excel dashboard analyzing Netflix content trends, distribution, and growth patterns from 2000–2021.

---

## 📊 Project Overview

This project presents a fully interactive Netflix analytics dashboard built in Microsoft Excel.  
It provides insights into content growth, show type distribution, ratings breakdown, and release trends over time.

The dashboard enables dynamic filtering using slicers and a timeline to explore content patterns efficiently.

---

## 🎯 Key Insights

- Total Titles: 8,804
- TV Shows: 6,129
- Movies: 2,675
- Rapid content growth observed after 2015
- TV-MA and TV-14 dominate content ratings
- Medium duration content is most common

---

## ⚙️ Features

- KPI Cards (Total Titles, TV Shows, Movies)
- Year Timeline Filter
- Show Type Slicer
- Duration Category Slicer
- Content Growth Trend
- Movie vs TV Show Release Comparison
- Rating Distribution Analysis
- Duration Distribution Analysis
